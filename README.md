## How to run this application on your local machine

1. Open terminal / command prompt in your desired folder

2. Run the following command

### `git clone https://github.com/lilyLILYLL/Resonate_contacts_app.git`

3. Then navigate into the project's root folder

### `cd contacts-app`

4. Install dependencies

### `npm install`

5. Start the application

### `npm run dev`
